Open the "main.py" file and run the project.

Install libraries if needed

pip install opencv-python
pip install numpy
pip install natsort

You can create a folder for collecting images from which the sprite will be collected,
if you create a folder with any name in the "SourceImages" directive,
then upload the images there.

You can add images that will be converted into a sprite.
Upload them to the folder "ImagesToConvert"

After starting the program,
select a folder for collecting images from which the sprite will be collected.

Select an image to be converted into a sprite.

After that you can get the result in "ConvertedImages" folder
